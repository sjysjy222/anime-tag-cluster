# anime-tag-cluster

数据来自https://bangumi.tv/anime
基于UMAP算法与标签标记数的泛百合动画聚类图。
使用Anime下拉菜单可选择具体动画及其相邻的10部动画。
使用Tag下拉菜单可选择具体动画标签。
建议使用PC端查看。

代码位于Bangumi_tag_umap.ipynb

在release中可以找到所有动画，及百合动画聚类的源文件
